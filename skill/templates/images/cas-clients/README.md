# /images/cas-clients/

Déposer ici les images des 3 cas clients du portfolio :

- `cas1.png` (ou `.jpg`) — cover ou mockup du cas 1
- `cas2.png` (ou `.jpg`) — cover ou mockup du cas 2
- `cas3.png` (ou `.jpg`) — cover ou mockup du cas 3

Le fichier `texte-cas-clients.txt` (à côté) contient les **textes associés** : titre, secteur, année, brief original, ce qui a été livré. Le skill apparie automatiquement chaque image avec son texte correspondant (cas1.png ↔ Cas 1 du `.txt`).

Si une image manque, le skill génère un mockup CSS éditorial à sa place. Si le `.txt` manque, le skill propose 3 cas-types inventés cohérents avec le brand.
